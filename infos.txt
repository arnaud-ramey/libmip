http://doc.ubuntu-fr.org/bluetooth#problemes_connus :
arnaud@eeepc:~$ sudo rfkill unblock all
arnaud@eeepc:~$ sudo hciconfig hci0 up

arnaud@eeepc:~$ sudo hcitool -i hci0 lescan
D0:39:72:B7:AF:66 (unknown)
D0:39:72:B7:AF:66 Bubi

== Connecting to MiP ==

arnaud@eeepc:~$ sudo gatttool -i hci0 -b D0:39:72:B7:AF:66 -I
> connect

Get handles:
http://www.jaredwolff.com/blog/get-started-with-bluetooth-low-energy/

> char-desc

Read LED:
> char-read-hnd 0x83
